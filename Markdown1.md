## Списки

Чтобы добавить не нумерованные списки, необходимо пункты выделить (*). Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимо списки просто пронумеровать. Например:

1. Элемент 1
2. Элемент 2
3. Элемент 3
